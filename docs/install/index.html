Binary packages

You can download the latest binary package from here or from the GitHub release page.

1.) Download the latest version: reno-linux-x86.zip

2.) Run it: ./reno

1.) Download the latest version: reno-0.1.0.jar

2.) Run it: java -jar reno-0.1.0.jar
