```text
Usage:
    reno dump
    reno update

reading notes extractor

Options and flags:
    --help
        Display this help text.
    --version, -v
        Print the version number and exit.

Subcommands:
    dump
        Dump the annotations from the given Pdf
    update
        update the notes in the Org-file with new annotations from the Pdf
```
